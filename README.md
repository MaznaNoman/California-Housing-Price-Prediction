# California Housing Price Prediction using ANN

This project uses an Artificial Neural Network (ANN) built with TensorFlow and Keras to predict California housing prices based on housing and demographic features.

## Technologies Used
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn

## Features
- Median Income
- House Age
- Average Rooms
- Average Bedrooms
- Population
- Average Occupancy
- Latitude
- Longitude

## Model Architecture
- Dense Layer (64, ReLU)
- Dense Layer (32, ReLU)
- Output Layer (1)

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)

## Project Workflow
1. Data Preprocessing
2. Feature Scaling
3. Train-Test Split
4. ANN Model Training
5. Model Evaluation

## Dataset
California Housing Dataset
